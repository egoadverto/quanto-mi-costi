# Roadmap Tecnica

## MVP (completato)
- Veicoli CRUD
- Rifornimenti CRUD
- Spese CRUD
- Dashboard base

## Step 2 (completato)
- Filtri avanzati (veicolo, data inizio/fine, categoria)
- Report migliorati (barre orizzontali, andamento mensile, efficienza)
- Grafici semplici senza librerie

## Step 3 (completato)
- Export CSV (veicoli, rifornimenti, spese)
- Import CSV (veicoli, rifornimenti, spese)
- Backup JSON (download completo)
- Ripristino JSON (import backup)

## Non fare
- AI
- integrazioni esterne
- notifiche
- multi-user

## Note
Tutte le funzionalità principali per un app personale di tracciamento costi veicoli sono ora implementate.
